# Test-IT-Workin

Реализованно следующее:

1. Создана страница с заголовком "User Management".

2. На странице имеется форма для добавления нового пользователя, состоящая из следующих полей:

        Имя (текстовое поле)

        Фамилия (текстовое поле)

        Email (текстовое поле)

Кнопка "Добавить" для отправки данных формы.

3. Под формой таблица, отображающая список пользователей.

4. Возможность удаления пользователей из списка.

5. Валидация полей формы (проверка на заполненность и правильный формат email).

6. Сохранение списка пользователей в локальное хранилище браузера.

Использовал:

1. React create App
2. JavaScript
3. Redux
4. HTML
5. CSS-модули

Для запуска необходимо:

1. Склонировать репозиторий
2. "Провалиться" в папку Client
3. Ввести команду - npm i
4. Далее ввести команду - npm start
5. При необходимости открыть браузер: http://localhost:3000/
